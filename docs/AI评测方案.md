# AI评测方案设计
## 1. 评测目标
验证模型对JD-简历匹配的判断准确性，定位Prompt缺陷、打分偏差
## 2. 评测数据集
CSV格式，字段：resume（简历文本）
## 3. 量化指标
1. match_score：综合匹配度 0-100
2. missing_count：简历缺失核心能力数量
3. ai_match：AI相关经历匹配度 0-10
## 4. 评测流程
1. 导入统一JD标准
2. 批量遍历全部简历自动打分
3. 导出csv结果，人工抽样校验
4. 根据低分样本优化Prompt模板
## 5. 迭代逻辑
收集Bad Case，更新Prompt，重复评测，提升匹配准确度
